# hide-and-show-password
A lightweight application to show and hide passwords with a single click, enhancing user experience and privacy.

Live Demo:
https://github.com/sahu-jagrati/hide-and-show-password

Features:
👁️ Show/Hide password with a single click
🔒 Improves user privacy and input accuracy
⚡ Lightweight and fast
🎨 Clean and minimal UI
